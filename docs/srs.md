심부름 서비스 (가칭)
=======

이 문서는 심부름 서비스 (가칭)의 공동 개발을 위한 요구사항 수집 문서입니다. 


개요
* 심부름 서비스는 심부름 요청자가 심부름 제공자에게 약간의 보상과 함께 간단한 부탁을 쉽게 할 수 있도록 합니다. 
* 심부름은 간단한 질문에 대한 대답이나 카풀 요청, 가벼운 물건 배달 등을 포함할 수 있습니다. 
* 심부름 서비스 이용을 위해서는 사용자의 기본적인 연락처, 위치 등을 공유해야 하고 요청자가 제공자들의 정보를 검색해 심부름을 부탁합니다.
* 보상은 비트코인과 같은 디지털 화폐로 이뤄질 수 있으며 부담되는 내용을 제한하고자 한화 5천원 이하의 작은 금액으로 제한합니다.
* 요청자의 요청에 응답하여 심부름계약이 이뤄지면 요청자가 제공하는 보상은 스마트 계약으로서 작성되며 정해진 시간 내에 심부름이 수행되고 요청자가 심부름이 완료됨을 확인하면 보상을 확정합니다. 
* 심부름이 적절히 완료되지 않는 경우 서비스에서 내용을 확인하여 요청자에게 보상을 반환합니다. 


앱 요구사항
* 비트코인 주소
* 심부름제공자
* 푸시 수신
* 안드로이드 단말 지원


서버 요구사항
* OpenAPI
* 사용자 정보 저장, 변경, 삭제
* 사용자 위치 저장, 검색
* 푸시 전송
* 사용자간 심부름 계약 생성, 종료, 삭제
* 사용자간 비트코인 전송
